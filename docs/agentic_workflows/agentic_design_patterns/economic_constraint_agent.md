# Economic Constraint Agent (agentic design pattern)

**Economic Constraint Agent** is an agentic *design pattern*: a reusable way to **optimize under explicit
budgets**—cost, time, carbon—so trade-offs are stated in the same currency operations and finance already argue
in—before you pick a concrete product story (auction, hiring, logistics, etc.).

In this layout, **constraint models** version coefficients; decision agents query them before committing routes or
spend, and outputs show **Pareto fronts** instead of fake single-objective perfection.

When **carbon budgets** bind, Coffee Agntcy picks **greener shipping** with eyes open about days and dollars.
