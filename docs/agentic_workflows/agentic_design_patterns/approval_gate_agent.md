# Approval Gate Agent (agentic design pattern)

**Approval Gate Agent** is an agentic *design pattern*: a reusable way to **pause the graph for a human** when stakes
exceed autonomy—bulk buys, novel counterparties, exceptions that would embarrass the board if automated quietly—before
you pick a concrete product story (auction, hiring, logistics, etc.).

In this layout, the workflow **packages context** (what, why, policy, deltas) and waits for **non-repudiable approval**
before irreversible side effects resume.

**High-risk bulk purchases** at Coffee Agntcy earn an executive line—not because agents are stupid, but because
accountability still has a surname.
